Step 2 : Install TexMaker (Recomended) Or use TexWorks </br>
Step 3 : Create a folder, Download and unzip the files to your folder </br>
step 4 : Open main.tex in TexMaker </br>
step 5 : Edit contents, Enjoy </br>
